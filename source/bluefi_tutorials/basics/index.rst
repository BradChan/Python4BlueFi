====================
入门级教程
====================

从这里开始使用，你将开始学习BlueFi的IoT应用编程，并逐步地掌握Python编程语言。学习进度安排由简单到复杂，只需要你具备
能够用电脑编辑文本和基本的编程思维能力。编程思维能力要求有多高呢？你需要认识以下短语，并能想象得出计算机执行每一条短
语时的行为：

  - if .. else ..
  - while ..
  - for ..
  - sum=100
  - print("hello world")
